
# Rohan Shende — Portfolio

A concise, professional single‑page portfolio showcasing Rohan Shende — a final‑semester B.Com student and aspiring Data Analyst. This site highlights core skills, real project work, and direct contact links, presented with a modern, responsive design and subtle visual polish.

## Overview

This repository contains a personal portfolio website built to introduce Rohan's background, technical skills, and sample projects. The page is designed for recruiters, collaborators, and anyone seeking to learn about Rohan's capabilities and professional interests.

## Site Structure

- **Hero** — Profile avatar, short headline, and a prominent "Hire Me" call-to-action.
- **About** — Brief professional bio and personal interests.
- **Skills** — Visual skill bars for SQL, HTML, CSS, Python, and Excel.
- **Projects** — Project cards demonstrating database and application work (Ecommerce SQL, Hospital Management, Coaching Institution).
- **Contact** — Simple contact form plus direct email and LinkedIn links.
- **Footer** — Copyright and quick links.

## Key Features

- Clean, responsive layout with accessible navigation.
- Animated avatar and subtle hover effects for a modern aesthetic.
- Skill proficiency bars for quick visual assessment.
- Project showcase with images and short descriptions.
- Contact section with mailto link and LinkedIn profile for networking.

## Technologies

- HTML5
- CSS3 (responsive layout, animations)
- JavaScript (for small UI behaviors in `script.js`)
- External images hosted via postimg and local `blobs/` folder for assets

## Usage

To preview locally, open `index.html` in a browser. For deployment, host the folder on any static web host (GitHub Pages, Netlify, Vercel, etc.).

To modify content or visuals:

- Edit site markup in `index.html`.
- Update styles in `style.css`.
- Adjust interactivity in `script.js`.
- Replace images in `blobs/` or update external image URLs in the HTML.
